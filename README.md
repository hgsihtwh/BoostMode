# BoostMode - Formula 1 Mobile App

Android-приложение для отслеживания сезона Формулы 1 2026: карта гонок, расписание, предсказания подиума с реальными результатами.

## Требования

- JDK 17+
- Android SDK (API 26+)
- Gradle 8+

## Сборка APK через командную строку

### 1. Клонировать репозиторий

```bash
git clone https://github.com/hgsihtwh/BoostMode.git
cd BoostMode
```

### 2. Собрать debug APK

```bash
./gradlew assembleDebug
```

На Windows:
```bash
gradlew.bat assembleDebug
```

### 3. Найти собранный APK

После успешной сборки APK находится по пути:
```
app/build/outputs/apk/debug/app-debug.apk
```

### 4. Установить APK на устройство

```bash
adb install app/build/outputs/apk/debug/app-debug.apk
```

## Сборка release APK

```bash
./gradlew assembleRelease
```

APK будет по пути: `app/build/outputs/apk/release/app-release-unsigned.apk`

## Руководство пользователя

### Карта гонок

1. На главном экране нажмите на экран - откроется меню
2. Выберите **Map** - откроется интерактивная карта мира со всеми гонками сезона F1 2026
3. Управление картой: перемещение - свайп пальцем, масштаб - щипок двумя пальцами
4. Нажмите на любой маркер на карте - появится всплывающее окно с названием Гран-при, номером раунда и городом
5. Нажмите на всплывающее окно - откроется экран трассы с интерактивной картой, приближённой к гоночному треку, а также информацией: рекордное время круга, количество поворотов, число кругов

### Предсказания и результаты гонок

1. В меню выберите **My Predictions**
2. Гонки разделены на три группы: **previous** (завершённые), **current** (текущая), **upcoming** (предстоящие)
3. Для любой гонки нажмите на поле 1 / 2 / 3 в колонке "my prediction" - выберите пилота из списка
4. Для завершённых гонок в колонке **"reality"** автоматически отображаются реальные результаты топ-3, полученные из Jolpica F1 API
5. Сравните свои предсказания с реальными итогами гонки
